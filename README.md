# mi_sds_seedtrt

## Overview

This repository contains the annotated analytical workflow used for the Michigan-only longitudinal soybean sudden death syndrome (SDS) seed treatment manuscript.

Working manuscript title:

> **Disentangling pathogen pressure, disease expression, and yield loss in soybean sudden death syndrome in Michigan**

The workflow integrates:

- Longitudinal SDHI seed treatment analyses
- Mixed-effects modeling
- Root disease index (RDX) analyses
- PCA and multivariate analyses
- Manuscript figure generation
- Supplemental table workflows
- Michigan trial-location mapping
